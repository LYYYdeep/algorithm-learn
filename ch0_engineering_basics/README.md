# Chapter 0: Engineering Basics

本目录用于记录算法工程师实习准备中的基础工程能力笔记。

## Files

- `01_linux_shell_notes.md`：Linux / Shell 常用命令和服务器训练相关操作
- `02_git_notes.md`：Git / GitHub 基础工作流
- `03_math_notes.md`：机器学习相关数学基础

## Learning Goal

掌握算法实习中最常用的工程基础：

- 能在 Linux / macOS 终端中完成文件、目录、搜索、进程、日志查看等操作
- 能使用 Git 管理代码版本并上传 GitHub
- 能理解机器学习中最常见的线性代数、概率统计、微积分和优化概念
